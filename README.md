# Credit Scoring Prediction System

## Overview

This project predicts whether a loan should be approved or rejected based on a customer's financial information using Machine Learning.

## Features

* Predicts loan approval status
* User-friendly web interface
* Flask backend API
* Machine Learning model using Logistic Regression

## Technologies Used

* HTML
* CSS
* JavaScript
* Python
* Flask
* Scikit-Learn
* Pandas

### Backend

cd backend

python app.py

### Frontend

Open index.html in a browser or run using Live Server.
